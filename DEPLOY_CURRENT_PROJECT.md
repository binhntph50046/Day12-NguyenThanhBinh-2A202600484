## 📝 BƯỚC 2: Deploy Lên Render

### **2.1. Sign Up Render**

1. Vào: https://render.com
2. Click **"Get Started"**
3. Sign up với **GitHub account**
4. Authorize Render to access GitHub

### **2.2. Create New Blueprint**

1. Click **"New +"** (góc trên bên phải)
2. Select **"Blueprint"**
3. Connect GitHub repository: `Day12-NguyenThanhBinh-2A202600484`
4. Render sẽ tự động detect file `render.yaml`
5. Review services:
   - ✅ `nguyen-thanh-binh-agent` (Web Service)
   - ✅ `redis` (Redis Database)

### **2.3. Configure Environment Variables**

Render sẽ tự động set hầu hết variables, nhưng bạn cần check:

**Click vào service → Environment tab:**

```bash
# Đã tự động set:
PORT=8000
ENVIRONMENT=production
DEBUG=false
REDIS_URL=<auto-generated>
RATE_LIMIT_PER_MINUTE=5
DAILY_BUDGET_USD=5.0

# Cần generate:
AGENT_API_KEY=<auto-generated>
```

**Generate API Key mạnh hơn (optional):**

```powershell
# PowerShell
-join ((48..57) + (65..90) + (97..122) | Get-Random -Count 32 | % {[char]$_})
```

Copy key và update trong Render.

### **2.4. Deploy**

1. Click **"Apply"**
2. Render sẽ bắt đầu:
   - ✅ Provision Redis (~30 giây)
   - ✅ Build Docker image (~2-3 phút)
   - ✅ Deploy service (~1 phút)

**Đợi cho đến khi thấy:**
```
✓ Redis: Live
✓ nguyen-thanh-binh-agent: Live
```

### **2.5. Get Public URL**

1. Click vào service `nguyen-thanh-binh-agent`
2. Copy URL ở đầu trang:
   ```
   https://nguyen-thanh-binh-agent.onrender.com
   ```

## 🧪 BƯỚC 3: Test Production

### **3.1. Test Health Check**

```powershell
# Set URL (thay bằng URL của bạn)
$url = "https://nguyen-thanh-binh-agent.onrender.com"

# Test health
curl "$url/health"
```

**Expected response:**
```json
{
  "status": "ok",
  "version": "1.0.0",
  "environment": "production",
  "uptime_seconds": 12.3,
  "total_requests": 1,
  "checks": {
    "llm": "mock"
  },
  "timestamp": "2026-04-17T10:30:00Z"
}
```

### **3.2. Get API Key**

```powershell
# Trong Render Dashboard → Service → Environment
# Copy giá trị của AGENT_API_KEY
```

### **3.3. Test API Endpoint**

```powershell
# Set variables
$url = "https://nguyen-thanh-binh-agent.onrender.com"
$key = "<paste-api-key-here>"

# Test API
curl -X POST "$url/ask" `
  -H "Content-Type: application/json" `
  -H "X-API-Key: $key" `
  -d '{"question": "Hello from production!"}'
```

**Expected response:**
```json
{
  "question": "Hello from production!",
  "answer": "Đây là câu trả lời từ AI agent (mock)...",
  "model": "gpt-4o-mini",
  "timestamp": "2026-04-17T10:30:00Z"
}
```

### **3.4. Test Rate Limiting**

```powershell
# Make 6 requests quickly
for ($i=1; $i -le 6; $i++) {
    Write-Host "Request $i"
    curl -X POST "$url/ask" `
      -H "Content-Type: application/json" `
      -H "X-API-Key: $key" `
      -d "{`"question`": `"Test $i`"}"
    Start-Sleep -Milliseconds 100
}
```

**Expected:** Request 6 sẽ nhận `429 Too Many Requests`

## 🆘 NẾU GẶP LỖI

### **Lỗi 1: Git push failed**

```powershell
# Check remote
git remote -v

# Remove và add lại
git remote remove origin
git remote add origin https://github.com/yourusername/Day12-NguyenThanhBinh-2A202600484.git
git push -u origin main
```

### **Lỗi 2: Render build failed**

1. Check logs trong Render Dashboard
2. Verify `render.yaml` có đúng path:
   ```yaml
   dockerfilePath: ./06-lab-complete/Dockerfile
   dockerContext: ./06-lab-complete
   ```

### **Lỗi 3: Health check failed**

1. Check logs: Render Dashboard → Logs
2. Verify PORT=8000 trong environment variables
3. Wait 2-3 phút cho service khởi động
